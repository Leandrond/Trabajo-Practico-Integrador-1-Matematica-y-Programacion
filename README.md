Trabajo Práctico Integrador 1 — Matemática & Programación I
Descripción General:
Este repositorio contiene el desarrollo del primer Trabajo Práctico Integrador para la carrera de Tecnicatura Universitaria en Programación (UTN FRVM)
El proyecto busca articular los fundamentos matemáticos de las Unidades 1, 2 y 3 con la implementación lógica en Python, abordando problemáticas reales del ámbito informático

Ejes Temáticos Abordados:
Unidad 1 — Conjuntos y Lógica: Validación de usuarios y análisis de consistencia mediante operaciones de conjuntos y tablas de verdad

Unidad 2 — Funciones: Análisis y optimización de modelos de costos de desarrollo (lineales y cuadráticos)

Unidad 3 — Matrices: Evaluación del rendimiento de sistemas distribuidos y detección de cuellos de botella mediante álgebra matricial

Tecnologías y Criterios Técnicos:
Lenguaje: Python (utilizando exclusivamente estructuras básicas como condicionales, ciclos, vectores y matrices para demostrar lógica de programación)

Control de Versiones: Git (con un registro de commits significativos que reflejan el avance progresivo del equipo)

Documentación: El proyecto incluye un informe técnico en PDF y la defensa oral mediante video
